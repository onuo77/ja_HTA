사원관리
*새 사원 등록폼에서 직종, 매니저,소속부서는 데이터베이스에서 직종, 매니저, 부서정보를 조회해서 select 태그로 만들어서 선택하게 할 것
*사원아이디, 입사일은 입력받지 말것
*사원아이디는 employees_seq로 증가된 값을 이용하고, 입사일은 sysdate로 현재 날짜가 저장되도록 함
	emp/main.jsp
		- 주요 메뉴
			전체 사원 목록
			사원정보 검색
			새 사원 등록
			사원정보 변경
			사원정보 삭제
	emp/list.jsp
		- 전체 사원 목록 제공(전체 사원들 목록 페이지)
		- Dao(List 객체 생성, SQL작성, dbms연결, while문으로 출력 후 추가, dbms종료, 생성한 객체 반환)
	emp/search.jsp
		- 사원 정보 검색 결과 제공
	emp/detail.jsp
		- 사원 상세정보 제공(사원 목록 페이지에서 사원 이름 클릭했을 때 사원 상세정보 나오는 페이지)
		- Title 인사관리 :: 사원관리
		- h1 사원 상세 정보
		- p 사원의 기본정보를 표시하고, 그 사원의 사원 정보를 표시한다.
		- 요청객체에서 요청파라미터 값 조회
		- h3 사원 기본 정보
		- dl, dt, dd (사원아이디, 사원이름, 이메일, 전화번호, 입사일, 직종아이디, 직종명, 급여, 커미션, 매니저아이디, 매니저이름, 부서아이디, 부서명)
		- Dto : employeeDto로 반환하는 getEmployeeDto(int empId) 메소드 생성
		- sql, dto 객체 null로 지정, dbms 연결, ?값지정, if(), dbms 삭제, return dto
	emp/form.jsp
		- 새 사원정보 등록 폼 제공(새 사원 등록했을 때 사원정보 입력펌이 나오는 페이지)
		- 인사관리 :: 사원관리
		- h1 새 사원 등록 페이지
		- p (사원아이디, 사원이름, 이메일, 전화번호, 입사일, 직종아이디, 직종명, 급여, 커미션, 매니저아이디, 매니저이름, 부서아이디, 부서명)
		  입력하고, 새 사원을 등록하세요
		- 직종, 매니저, 소속부서 Db에서 불러오기
		- h3 새 사원 등록폼
		- form - label 
	emp/add.jsp
		- 새 사원정보 등록(사원정보 입력폼의 값을 제출받아서 새 사원을 등록하고 리스트 페이지로 리다이렉트 시키는)
